# Product System (OOP)

Aplicação em JavaScript que demonstra conceitos de programação orientada a objetos, como classes, herança e sobrescrita de métodos, com exibição dinâmica dos dados na tela.

## 🚀 Funcionalidades
- Criação de produtos com classe base
- Extensão de classe para eletrônicos (herança)
- Sobrescrita de método para adicionar garantia
- Exibição dinâmica dos dados no DOM

## 🛠 Tecnologias
- HTML
- CSS
- JavaScript
